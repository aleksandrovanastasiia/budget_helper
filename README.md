# 7. Командный проект по дисциплине «Основы программирования»
## Тема: Бюджетный помощник

Программа хранит ежедневные расходы пользователя за месяц и позволяет:
- добавить расход: день, сумма, категория;
- получить сумму расходов за период с дня A по день B;
- найти день с максимальным расходом;
- отсортировать категории по сумме трат методом вставок;
- отменить последний добавленный расход через стек;
- хранить траты в дереве поиска;
- сохранять и загружать данные из файла.

## Команда
- Лера
- Настя

**Лера**
Структуры данных

**Настя**

Префиксные суммы
Поиск и сортировка
Пользовательский интерфейс

## Как запустить
1. Установите Python 3.10+.
2. Откройте папку проекта.
3. Запустите:
```bash
python main.py
```
